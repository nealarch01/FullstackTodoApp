## Tech Stack
### Frontend (iOS app):
- Swift
- SwiftUI
- Model-View-ViewModel (MVVM)

### Backend (REST API):
[Postman Documentation](https://documenter.getpostman.com/view/21072555/2s935hPmYK)
- TypeScript
- Express.js REST API
- JSONWebTokens (JWTs)
- Model-View-Controller (MVC)

### Database:
- PostgreSQL 15.0
