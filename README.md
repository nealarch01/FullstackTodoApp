## Tech Stack
### Frontend (iOS app):
- Swift
- SwiftUI
- Model-View-ViewModel (MVVM)

### Backend (REST API):
- TypeScript
- Express.js REST API
- JSONWebTokens (JWTs)
- Model-View-Controller (MVC)

### Database:
- PostgreSQL
